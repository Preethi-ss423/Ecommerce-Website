# 🛍️ Preethi Store – E-Commerce Website

A responsive and interactive E-Commerce website developed using **HTML, CSS, and JavaScript**. The project simulates an online shopping experience with features such as product browsing, shopping cart management, user authentication, search, and category filtering.

---

## 🚀 Features

- 🏠 Responsive Home Page
- 📦 Product Listing
- 🔍 Search Products
- 🏷️ Category Filter
- 🛒 Add to Cart
- ➕ Increase Quantity
- ➖ Decrease Quantity
- 🗑️ Remove from Cart
- 💰 Dynamic Total Price Calculation
- 💾 Local Storage Support
- 👤 User Registration
- 🔐 User Login
- 🚪 Logout
- 🔢 Cart Item Counter
- 📱 Responsive Design
- 🎨 Modern UI

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API

1. Download or clone this repository.
2. Open the project folder in Visual Studio Code.
3. Open `index.html` using Live Server.
4. Start exploring the website.
